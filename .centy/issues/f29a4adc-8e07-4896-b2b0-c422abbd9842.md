# Add configurable number of virtual screens (1 to 6)

## Description

Allow the user to configure how many virtual screens are displayed, from a minimum of 1 up to a maximum of 6. Currently the app always creates 6 virtual displays. This setting should let users choose how many they want.

## Acceptance Criteria

* [ ] A new setting is available to configure the number of virtual screens
* [ ] The setting accepts values from 1 to 6
* [ ] The configured number of screens is applied when creating virtual displays
* [ ] The setting is accessible from the macOS menu bar settings UI (see issue #3)
* [ ] The setting persists across app sessions
* [ ] Input is validated to enforce the 1-6 range
