# Add names for virtual displays

Virtual displays should have user-friendly names to make them easier to identify and manage. Each virtual display should be named based on its position in the grid layout:

* **Left**
* **Middle**
* **Right**
* **Top**
* **Bottom**

These names should replace or supplement the current identification by resolution or index.
