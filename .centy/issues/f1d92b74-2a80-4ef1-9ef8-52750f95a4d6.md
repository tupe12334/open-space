# Move distance setting to macOS menu bar

## Problem

The stage distance setting is currently configured via an in-app overlay UI (toggled with Tab, adjusted with Arrow Up/Down). We want it accessible from the macOS menu bar instead.

## Current behavior

* Press **Tab** to open a settings panel inside the app window
* Use **Arrow Up / Arrow Down** to adjust distance
* Settings are persisted to `settings.json`

## Desired behavior

Move the distance configuration to a native macOS menu bar item so users can adjust it from the top bar without an in-app overlay.

## Files involved

* `src/settings.rs` — current settings UI and input handling
