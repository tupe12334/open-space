# Spawn app full screen on glasses screen

## Description

The Centy app should spawn in full screen mode on the glasses screen instead of the default windowed mode.

## Expected behavior

When the app launches, it should automatically detect the glasses display and open in full screen on that screen.

## Current behavior

The app opens in a regular window, requiring manual resizing/moving to the glasses screen.
