# Add settings configuration app for stage distance

## Description

Add an in-app settings screen that allows the user to configure application settings. 

## Initial Scope

For now, the settings app will only support one setting:

- **Distance to stage** — lets the user configure how far the virtual stage/screen is placed from the viewer.

## Acceptance Criteria

- [ ] A settings UI is accessible from within the app
- [ ] The user can view and modify the distance to stage value
- [ ] The configured distance is applied to the virtual stage placement
- [ ] The setting persists across app sessions
