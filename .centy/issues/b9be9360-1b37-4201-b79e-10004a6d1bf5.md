# Add settings configuration app for stage distance

## Description

Add a settings configuration interface accessible from the macOS top menu bar (not inside the app itself). This will be a menu bar item that lets the user configure application settings.

## Initial Scope

For now, the settings menu will only support one setting:

* **Distance to stage** — lets the user configure how far the virtual stage/screen is placed from the viewer.

## Acceptance Criteria

* [ ] A settings option is accessible from the macOS top navigation/menu bar
* [ ] The settings UI is separate from the main app window (menu bar based)
* [ ] The user can view and modify the distance to stage value
* [ ] The configured distance is applied to the virtual stage placement
* [ ] The setting persists across app sessions
