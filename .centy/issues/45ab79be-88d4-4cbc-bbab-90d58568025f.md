# Set up Dependabot for automated dependency updates

## Description

Configure GitHub Dependabot to automatically create pull requests when new versions of dependencies are available. This ensures dependencies stay up to date without manual tracking.

## Acceptance Criteria

* [ ] Create `.github/dependabot.yml` configuration file
* [ ] Configure Dependabot for the `cargo` (Rust) package ecosystem
* [ ] Set update schedule (e.g., weekly)
* [ ] Configure a reasonable limit on open PRs (e.g., 10)
* [ ] Optionally group minor/patch updates together to reduce PR noise
* [ ] Verify Dependabot creates PRs after the configuration is merged to main
