# Add Husky Cargo pre-commit and pre-push hooks

## Description

Set up [cargo-husky](https://github.com/rhysd/cargo-husky) in the repository to enforce code quality checks via Git hooks.

## Hooks

### Pre-commit

* `cargo fmt --check` — ensure code is properly formatted
* `cargo clippy -- -D warnings` — catch common mistakes and lint issues

### Pre-push

* `cargo test` — run the full test suite before pushing

## Acceptance Criteria

* [ ] `cargo-husky` is added as a dev dependency
* [ ] Pre-commit hook runs `cargo fmt --check` and `cargo clippy`
* [ ] Pre-push hook runs `cargo test`
* [ ] Hooks are automatically installed when contributors run `cargo test` (cargo-husky default behavior)
* [ ] Document the hook setup in CONTRIBUTING.md
