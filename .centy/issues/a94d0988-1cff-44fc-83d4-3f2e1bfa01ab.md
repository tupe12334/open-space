# Update objc2 and objc2-foundation

## Description

Update the `objc2` and `objc2-foundation` crates to their latest versions. These are tightly coupled dependencies and should be updated together. This update may also resolve issue #2 (macOS 26 crash due to type encoding mismatch).

## Changes

* `objc2`: `0.5.2` -> `0.6.4`
* `objc2-foundation`: `0.2.2` -> `0.3.2`
