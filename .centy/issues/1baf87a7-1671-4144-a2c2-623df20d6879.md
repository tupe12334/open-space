# Webcam-Based Distance Estimation (Camera to AR Glasses)

## Context

Use the computer's webcam (FaceTime camera) to detect the XREAL Air AR glasses on the user's face and display the estimated distance between the camera and the glasses in the terminal via `info!()` logging.

The project already uses ObjC interop extensively (`objc2`, `core-media`, `core-video`, `dispatch2`). The `screen_capture` module provides a near-identical pattern for what we need: background thread capturing frames via an Apple framework, with a `declare_class!` delegate.

## Approach

Use **native macOS frameworks** (AVFoundation for webcam capture, Vision for face detection) via `objc2`/`msg_send!` — no new crate dependencies needed. Estimate distance using the **pinhole camera model** with face bounding box width from Apple's Vision framework.

**Distance formula:** `distance_cm = (KNOWN_FACE_WIDTH_CM * FOCAL_LENGTH_PX) / face_width_px`

## Implementation Steps

### Step 1: `types.rs` — Shared State
- `DistanceStore` resource: `Arc<Mutex<Option<f32>>>` holding latest distance in cm
- Follows `ImuStore` pattern from `src/modules/hmd/tracking/types.rs`

### Step 2: `permission.rs` — Camera Permission
- `ensure_camera_permission()` function called before `App::new()`
- Uses `AVCaptureDevice` class via `AnyClass::get("AVCaptureDevice")` + `msg_send!`
- Check `authorizationStatusForMediaType:` with `AVMediaTypeVideo`
- Request access with `requestAccessForMediaType:completionHandler:` if needed
- Pattern: mirrors `src/modules/screen_capture/permission.rs`

### Step 3: `detection.rs` — Face Detection + Distance Calc
- Function: `detect_face_distance(pixel_buffer: &CVPixelBuffer, image_width: usize) -> Option<f32>`
- Create `VNImageRequestHandler` from the `CVPixelBuffer` via `msg_send!`
- Create `VNDetectFaceRectanglesRequest`
- Execute and read results -> `VNFaceObservation` bounding box
- Calculate distance: `(FACE_WIDTH_CM * FOCAL_LENGTH_PX) / (bbox_width * image_width as f32)`
- Constants: `FACE_WIDTH_CM = 15.0`, `FOCAL_LENGTH_PX = 900.0` (FaceTime 720p estimate)
- Apply exponential moving average smoothing (alpha ~0.3)

### Step 4: `mod.rs` — Webcam Capture
- `declare_class!` for `CameraDelegate` implementing `AVCaptureVideoDataOutputSampleBufferDelegate`
  - `captureOutput:didOutputSampleBuffer:fromConnection:` callback
  - Extract `CVPixelBuffer` from `CMSampleBuffer` (same as screen_capture)
  - Call `detect_face_distance()`, store result in `DistanceStore`
- `start_webcam_capture` startup system:
  - Spawn background thread
  - Get default video device: `AVCaptureDevice defaultDeviceWithMediaType:`
  - Create `AVCaptureDeviceInput`, `AVCaptureSession`, `AVCaptureVideoDataOutput`
  - Set pixel format to BGRA, set `alwaysDiscardsLateVideoFrames = YES`
  - Set delegate on dispatch queue
  - Start session
  - Keep thread alive with sleep loop
- `log_distance` update system:
  - Read `DistanceStore`, log distance via `info!()` (throttled to ~1 log/sec)

### Step 5: `plugin.rs` — Plugin Assembly
- `WebcamDistancePlugin` registers `DistanceStore` resource, `start_webcam_capture` (Startup), and `log_distance` (Update)

### Step 6: Integration
- `src/modules/mod.rs`: add `pub(crate) mod webcam_distance;`
- `src/main.rs`: add `ensure_camera_permission()` before `App::new()`, add `.add_plugins(WebcamDistancePlugin)`
- Link frameworks with `#[link(name = "AVFoundation", kind = "framework")]` and `#[link(name = "Vision", kind = "framework")]`
