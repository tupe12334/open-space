# Add center stage button to settings

## Problem

There is no way to quickly re-center the point of view (camera) in the app. Users need a convenient button to reset/center the stage view.

## Desired behavior

Add a **Center Stage** button to the settings UI (menu bar) that, when clicked, resets the camera/point of view to the default centered position.

## Acceptance criteria

* A "Center Stage" button is visible in the settings menu
* Clicking the button resets the point of view to the default center position
* The action should be immediate with no additional confirmation needed
